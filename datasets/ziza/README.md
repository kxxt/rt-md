# Ziza Dataset

We follow ibHH's partition scheme to partition
Ziza dataset into three parts,
first 10 hours for training, following 2 hours as peacetime,
and final 14 hours as validation set.

Please process the dataset following instructions from
open source code of [ibHH][ibHH]
and place the preprocessed CSV files here.

[ibHH]: https://github.com/akamai/Information-based-Heavy-Hitters-for-Real-Time-DNS-Exfiltration-Detection?tab=readme-ov-file#instructions
